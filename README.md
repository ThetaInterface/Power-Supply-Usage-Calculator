# Build
To build:
```
git clone https://github.com/ThetaInterface/Power-Supply-Usage-Calculator.git
cd Power-Supply-Usage-Calculator
chmod +x gradlew
./gradlew build
```
# Execute
To execute program:
```
cd build/libs
java -jar psuc-1.1.jar
```
# Requirements
Was tested on Arch Linux
```
OS: Linux
Java: jdk21

Plug off charging device if laptop
```
